# netrunner (GUI)

A [Zed **GPUI**](https://www.gpui.rs/) desktop front-end for netrunner. It runs
the shared [`netrunner-core`](../netrunner-core) engine and renders **live
download/upload bandwidth charts** while the test runs, plus a summary of ping,
throughput, connection quality, location, ISP and server.

```sh
cargo run -p netrunner
```
